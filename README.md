# Exercícios em Java

Repositório com exercícios básicos em Java focados em lógica de programação, entrada de dados e estruturas de controle.

## Tecnologias

* Java
* Scanner
* Random

## Arquivos

### Calculadora.java

Lê 5 números, calcula a soma e a média.

Conceitos:

* Loop `for`
* Acumulador
* Operações matemáticas

---

### GeradorNum.java

Gera um número aleatório dentro de um intervalo informado pelo usuário.

Conceitos:

* Classe `Random`
* Intervalo dinâmico
* Fórmula:

```
nextInt(max - min + 1) + min
```

---

### Login.java

Valida que usuário e senha não sejam iguais.

Conceitos:

* `do-while`
* Comparação de Strings (`equals`)

---

### NúmeroMaior.java

Lê 5 números e identifica o maior.

Conceitos:

* Loop `for`
* Comparação (`if`)
* Variável de controle (maior valor)

---

### Validação.java

Valida entrada de dados:

* Nome
* Idade
* Salário
* Sexo
* Estado civil

Conceitos:

* `do-while`
* Validação de dados
* `Character.toLowerCase()`

---

## Objetivo

Praticar lógica de programação e consolidar conceitos básicos em Java.
